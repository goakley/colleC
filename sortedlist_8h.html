typedef struct SortedList SortedList

A sorted list of values.

A list that will hold similar values in sorted order, smallest to largest. The size of the list's elements is determined on creation, as is the method of sorting them.

Creates a new SortedList.

Creates a new SortedList that holds elements of a specified size. The order of elements will be determined by the passed in comparison function.

Parameters
[in]sizeThe size of a single element
[in]comparThe comparator function, which takes two arguments that correspond to pointers to two items that could be in the list. If the first item goes before the second, the function should return <0. If the first item goes after the second, the function should return >0. If the order of the two items does not matter (equivalent), the function should return 0.
Returns
Pointer to an empty SortedList, NULL if a size of 0 was given or a NULL function pointer was given
